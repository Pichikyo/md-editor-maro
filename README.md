# md Editor MARO

A simple and modern Markdown editor that runs entirely in your browser — no installation required.

🔗 **Live:** https://pichikyo.github.io/md-editor-maro/

## Features

- **Split view** — Edit on the left, preview on the right in real time
- **3 themes** — Modern / Warm / Dark
- **Copy MD** — Copy raw Markdown to clipboard with one click
- **Table support** — Markdown tables render with clean styling
- **Zero setup** — Just open the HTML file in any browser

## Usage

Open `index.html` in your browser, or visit the live URL above.

Write Markdown on the left side. The preview updates instantly on the right.

## Local use

Download `index.html` and open it directly in your browser. No server needed.
